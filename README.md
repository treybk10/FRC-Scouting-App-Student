FRC SCOUTING APP!! 

This scouting app allows you to import a .mov or .mp4 video, enter the team number and alliance, and have AI scout the match for you! There's also stand scouting, alliance selection predictor, and picklist generator

This is the source code so you can make your own version! Please just enter your own Hack Club AI API key and thebluealliance.com API key.

This is built with streamlit and deployed through streamlit community cloud.
